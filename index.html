Welcome

This is the home of Pillow, the friendly PIL fork. PIL is the Python Imaging Library. If you've ever worried or wondered about the future of PIL, please stop! We are here to save the day.

Code

Our code is hosted on GitHub, tested on Travis CI and Landscape, and released on PyPI.

Documentation

Our documentation is hosted on readthedocs.org and includes installation instructions, guides, API reference, release notes and more.
